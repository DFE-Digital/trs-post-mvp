View and filter tasks across all services. Select tasks using the checkboxes, then use Batch assign to assign several at once. Use Manage allocation to see team workload.

What do the task statuses mean?
  • Not started — the task is in the queue and nobody has opened it yet.
  • In progress — someone is actively working on it.
  • On hold — work is paused (for example waiting for evidence, a dependency, or a scheduled callback). The task is not being progressed until that condition is met.
